# Fiber Buddy Web

A warm, mobile-first React web app for personal fiber tracking.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL Vite prints, usually:

```txt
http://localhost:5173
```

## Develop as mobile

In Chrome:

```txt
Right click → Inspect → Toggle device toolbar → choose Pixel 7
```

## Later mobile path

When the web app feels right, wrap it with:

- PWA first
- Then Capacitor for Android APK, if needed
